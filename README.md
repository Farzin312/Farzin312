### About

I lead engineering as CTO at one founding-stage startup and build solo as sole engineer at another. My work centers on Node.js, TypeScript, and PostgreSQL services that handle authentication, payments, and the operational complexity of real products. I integrate retrieval and ML where it earns its keep in the backend, not as a standalone experiment.

### Currently building

- **Sports Excitement (SPEX).** CTO and Lead Software Engineer. Owning system architecture, the backend API surface, hybrid VPS plus cloud infrastructure, and engineering standards across web and mobile.
- **Strictly Surf.** Sole engineer on a full-stack e-commerce and marketplace platform. Stripe payment flows with webhook reconciliation, ledger-based store credit and refunds, PostgreSQL with strict RLS.
- **Life OS.** A vault-native operating system for my own life and career engine. Routines, agents, and a knowledge graph driven against the Anthropic SDK. Public write-up at [farzinbuilds.com](https://www.farzinbuilds.com).

### Currently learning

- Distributed systems and applied ML for production backends.
- AWS Machine Learning Associate (exam mid-May 2026).
- M.Sc. Computer Science, AI Optimization at WGU, expected May 2026.
